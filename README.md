# Geometry Web

Браузерная копия Geometry Dash на чистом HTML5/Canvas — без сборки и зависимостей.

## Как играть

Открой **index.html** (версия для игроков).

- **Прыжок / полёт**: клик мышью, тап, `Пробел`, `↑` или `W` (удерживай — куб прыгает без остановки, кораблик и волна летят вверх)
- **Пауза**: `Esc` или кнопка ⏸ (меню на кнопках-иконках: заново, продолжить, практика-ромб, выход)
- **Практика** (ромб в паузе): `Z` — поставить чекпоинт, `X` — удалить последний. Чекпоинты также ставятся автоматически.

В игре: режимы **куб / самолётик / волна**, чёрные шипы и блоки, монеты, орбы (срабатывают и по клику, и при удержании — как в GD) и батуты, **ускорения** (0.8x/1x/1.5x/2x, работают во всех режимах), смена цвета фона, move-триггеры (двигают группы объектов), **сложность уровня** (лицо в стиле GD на карточке), разбивание кубика при смерти и эффект появления, счётчик попыток (каждый заход — с попытки 1), кнопка ℹ — статистика уровня.

**Карта прохождений** (кнопка-карта в лобби): космическая карта с узлами-уровнями, которые открываются по порядку — следующий узел после 100% предыдущего. Уровень 1 карты: **Pulsar**. Пройденные узлы получают звезду. Уровни карты живут в `js/map-levels.js` отдельно от обычного списка.

Работает на телефонах и планшетах: чёткая картинка на Retina, обводка текста с фолбэком для Safari, тач без задержек, автопауза при сворачивании вкладки, поддержка выреза iPhone, уважение к prefers-reduced-motion.

## Версия разработчика

Открой **dev.html** — то же самое плюс редактор (молоток) и карта правее него.

### Редактор

- Вкладки палитры: **Блоки** (4 вида), **Объекты** (шипы, монета, орбы, батуты), **Порталы** (куб/корабль/волна + 4 ускорения), **Триггеры** (цвет фона, move)
- **Строить** — ЛКМ ставит выбранный объект; **Двигать** — выбор и перетаскивание (рамкой — сразу несколько, `Shift` — добавить к выделению); **Удалять** — клик удаляет. В «Двигать/Удалять» палитра скрыта — видно, в каком ты режиме
- Инспектор справа у выделенного: **поворот** (⟲⟳ или `R`), **размер** (−/+), **группа** (номер для move-триггера), у цвет-триггера — цвет и время, у move-триггера — группа, сдвиг X/Y и время
- **Отмена/повтор** — стрелки слева или `Ctrl+Z` / `Ctrl+Y`
- **Копировать/вставить** — кнопки слева или `Ctrl+C` / `Ctrl+V`
- **Полоса внизу** — быстрое перемещение по уровню
- **Старт-поза** (вкладка «Триггеры») — зелёная клетка со стрелкой: тест ▶ начинается с ближайшей к камере старт-позы (режим куб/корабль/волна и ускорение подхватываются автоматически по пройденным порталам). Если старт-поз нет — тест идёт с левого края экрана
- **▶ (зелёная кнопка на канвасе)** — тест прямо в редакторе; `Esc` или ■ — назад
- **▶ Тест с начала** — полноэкранный прогон уровня целиком
- **Фон** — стартовый цвет фона; **♫ Музыка** — играет в редакторе и тестах, но **в экспорт не вшивается** (файл остаётся лёгким) — mp3 передаётся отдельно и кладётся в `music/`
- **⤓ Экспорт** — скачивает компактный `<название>.gwlevel.json`; **Импорт** — загружает обратно (старые файлы со вшитой музыкой тоже читаются)
- Спокойно тянет уровни на 10 000+ объектов

### Как добавить уровень в игру для игроков

1. Сделай уровень в редакторе и нажми **Экспорт** — получится файл `.gwlevel.json`
2. Передай этот файл — содержимое вставляется в `js/levels.js`:
   ```js
   window.LEVELS.push( /* сюда содержимое файла */ );
   ```
3. Уровень появится в списке уровней у игроков (листается стрелками).

## Кубики

Иконка кубика — `textures/cube1.jpg`. Если файла нет, рисуется стандартный зелёный кубик. Новые кубики добавляются в список `Icons.list` в `js/engine.js`.

## Запуск локально

Достаточно открыть `index.html` двойным кликом. Если браузер не даёт загрузить `cube1.jpg` с диска (политика file://), запусти любой локальный сервер, например:

```
python3 -m http.server 8000
```

и открой http://localhost:8000

## Структура

```
index.html      — игра (для игроков)
dev.html        — игра + редактор (для разработчика)
css/style.css   — стили интерфейса в духе GD
js/engine.js    — движок: физика, коллизии, отрисовка, звук
js/levels.js    — встроенные уровни (сюда вставляются экспортированные)
js/main.js      — лобби, гараж, выбор уровня, HUD
js/editor.js    — редактор уровней (только dev.html)
textures/       — картинки (кубики и будущие текстуры)
music/          — музыка уровней
```
